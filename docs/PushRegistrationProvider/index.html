register​For​Push(with​Device​Identifier:​locale:​callback:​) 

public func registerForPush(withDeviceIdentifier identifier: String,
                                locale: String,
                                callback: @escaping (PushRegistrationResult) -> Void)

Calls a push registration end point to register the given APNS device id.

This method stores the response on successful registration. Subsequent calls to this method with the same identifier bypass calls to the network and return the stored response in the completion handler. Calling this method with a different identifier will remove any stored response from storage.
