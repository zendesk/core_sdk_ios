get​Settings(_:​callback:​) 

public func getSettings<T>(_ type: T.Type, callback: @escaping (T.DefaultType) -> Void) where T: DecodableDefault

This method fetchs settings for a provided DecodableDefault type. Settings are cached for one hour. This method will return the cached settings or settings from the network depending on the validity of the cache.

Note: As a side-effect of this method, we update the ActionHandlerRegistry with the data we get from the cache or the network.
