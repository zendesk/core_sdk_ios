get​Machine​ID() 

public func getMachineID() -> String

Generates, stores and returns a UUID string to uniquely identify the host app install.

This identifier lives until the host app is deleted. It is generated and returned on first call of this API and then just returned on future calls.

Returns

The UUID to identify the install.
